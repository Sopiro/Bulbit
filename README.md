# Bulbit

Bulbit is a physically based raytracing renderer.

## Building
- Install [CMake](https://cmake.org/install/)
- Ensure CMake is in your system `PATH`
- Clone the repository 
  ```bash
  git clone https://github.com/Sopiro/Bulbit
  ```
- Build the project
  ```bash
  mkdir build
  cd build
  cmake ..
  cmake --build .
  ```
